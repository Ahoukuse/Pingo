# Pingo

### 项目介绍
为你浏览的网页添加评论区



### 安装教程

1. 点击本页面中的"克隆/下载",在弹出菜单中点击"下载ZIP"
2. 下载完成后解压文件
3. 打开Chrome。依次点开菜单>更多工具>扩展管理。在
扩展管理页面右上角，打开开发者模式。点击"加载已解压的扩展程序"，选择到刚刚下载的文件。

### 使用说明

1. 注册[Pingo](https://ahhhh.com.cn/pingo/signin "Pingo注册")。**PS:注册请不要使用您常用的密码。**
2. 在插件上登录。
